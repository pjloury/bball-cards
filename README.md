# 🏀 Hoops Elite — NBA Card Collection

A full-stack NBA trading card collecting web app. Open one pack of 5 cards per day, reveal them one by one, and build your collection. Cards feature real player stats pulled live from the NBA Stats API.

## Features

- **100 NBA players** with real career stats (via NBA Stats API — no key needed)
- **1 pack per day** — 5 cards per pack, enforced server + client side
- **4 rarities** — Common, Silver Chrome, Gold Chrome, Prismatic (with holographic CSS animations)
- **Pack opening animation** — tear it open, reveal cards one by one with sparkle effects
- **Card front** — Topps Chrome aesthetics: team color gradient, player photo, chrome border per rarity
- **Card back** — Bio + season-by-season career stats; league leader stats highlighted in red ★
- **Library** — Search, filter by rarity/position/team, sort, flip cards to view front & back
- **Collection export/import** — Back up and restore your collection as JSON

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Seed the database (fetches real stats from NBA API — takes ~3 min)
npm run seed

# 3. Start the server
npm start
```

Open **http://localhost:3000** in your browser.

## Tech Stack

| Layer | Tech |
|-------|------|
| Backend | Node.js + Express |
| Database | SQLite (better-sqlite3) |
| Player Stats | NBA Stats API (free, no key) |
| Player Photos | NBA CDN (free, no key) |
| Frontend | Vanilla HTML + CSS + JS (no build step) |

## Database

The SQLite database has two kinds of data:

| Table | Purpose | Persisted? |
|-------|---------|------------|
| `players` | 100 NBA players + career stats | Regenerated by `npm run seed` |
| `packs` | Pack opening history | Stored locally |
| `collection` | Your card collection | **Back up with Export button** |

The `db/hoops.db` file is in `.gitignore`. To restore player data on a fresh clone, run `npm run seed`.

**Your collection** is stored in both `localStorage` and the server DB. Use the **Export Collection** button in the app to download a `hoops-collection.json` backup you can import later.

## Re-seeding

The seed script hits `stats.nba.com` to pull real career stats for all 100 players. It includes polite rate limiting (600ms between requests). Takes about 3 minutes total.

```bash
npm run seed
```
